Instruções para efetuar um (push)
=================================

1. Utilizando o "git bash" acesse o diretório onde deseja criar o repositório;
2. digite ***git init***(Cria o repositório no diretório);
3. digite ***git add .***(Adiciona os arquivos existentes no diretorio para o sistema de versionamento);
4. digite ***git commit -m"Mensagem explicando as alterações do commit"***(Efetua o comit granvando a mensagem das alterações);
5. digite ***git remote add origin "http://endereço co repositório no github.git"***(adiciona a conexão com o repositório remoto);
6. digite ***git push origin master*** (Envia os dados do repositório local para o repositório remoto).
